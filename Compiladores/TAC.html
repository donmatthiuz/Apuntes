Instrucciones

Analiza el siguiente cdigo en TypeScript:

function calculate(x: number, y: number, z: number): number {
  return x * z + y * z - (z + y);
}

Representa este cdigo utilizando:

  • Una versin en cdigo de tres direcciones (TAC)
  • Una tabla de cudruplos
  • Una tabla de tripletas

Valida que el cdigo original y su representacin intermedia produzcan el mismo resultado con valores de prueba, por

ejemplo: x = 2, y = 3, z = 4

Solucion

Cdigo de 3 direcciones

function calculate: T1 = x * z

T2 = y * z

T3 = z + y

T4 = T1 + T2

T5 = T4 - T3

return T5

param x param y param z

T6 = call calculate, 3
